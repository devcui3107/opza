Opza.js

Thư viện modal tối ưu dành cho nhà phát triển. Sạch sẽ, hiện đại và rất nhẹ.

Cài đặt

Giới thiệu

Opza.js là một thư viện JavaScript thuần (Vanilla JS) được thiết kế đặc biệt để hỗ trợ việc xây dựng và quản lý các modal (hộp thoại bật lên) một cách nhanh chóng, dễ dàng, và linh hoạt.

Đặc điểm nổi bật

  • Nhẹ và nhanh: Chỉ 2KB, không cần thư viện phụ thuộc.
  • Tích hợp linh hoạt: Hoạt động tốt với mọi dự án và framework.
  • Tùy biến toàn diện: Kiểm soát hành vi và hiệu ứng modal dễ dàng.
  • CSS tự định nghĩa: Tự do thiết kế giao diện theo ý muốn.
  • Thân thiện với lập trình viên: API rõ ràng, tài liệu chi tiết.

Trường hợp sử dụng với templateId (không sử dụng content - chuỗi HTML trực tiếp)

html 
<template id="modal-1"> <!-- Tự css vào các class tự định nghĩa -->
  <div class="modal__header">Tiêu đề Modal 1</div>
  <div class="modal__body">
    <p>Nội dung Modal với TemplateId</p>
  </div>
</template>
javascript 
const modal = new Opza({
  templateId: "modal-1", // Chú ý id phải trùng với id của thẻ template tương ứng bên trên
});

Trường hợp sử dụng với content (không sử dụng templateId)

javascript 
const modal = new Opza({
  content: `<div>Nội dung content của modal<br>
  <a href='#!'>Click me!</a>
  </div>`, // Chuỗi HTML tuỳ ý (có thể tạo biến html để chứa rồi truyền biến vào)
  cssContainer: ["modal__content", "class-N"], // Nhận mảng hoặc chuỗi
});

Trường hợp bật tuỳ chọn destroy (xoá modal khỏi DOM sau khi đóng modal)

javascript 
const modal = new Opza({
  content: `<div>Nội dung content của modal<br>
  </div>`,
  cssContainer: ["modal__content", "class-N"],
  destroy: true, // Bật tuỳ chọn xoá Modal khỏi DOM khi đóng modal
});

Trường hợp tuỳ chọn cách đóng modal (Mặc định có 3 cách:)

  • Bấm nút Escape
  • Bấm vào vùng bên ngoài modal (overlay)
  • Bấm nút close (góc trên bên phải modal)
javascript 
const modal = new Opza({
  content: `<div>Nội dung content của modal</div>`,
  cssContainer: ["modal__content", "class-N"],
  closeTrigger: ["button", "Escape"], // Vô hiệu hoá overlay
});

Tuỳ chọn modal có nội dung Footer

javascript 
const modal = new Opza({
  content: `<div>Nội dung content của modal</div>`,
  cssContainer: ["modal__content", "class-N"],
  footerContent: "<div>Đây là nội dung footer</div>", // Thêm class để css trực tiếp

Method

Phương Thức Mô Tả
open() Mở modal
close() hoặc close(true) Đóng modal. Nếu truyền true thì modal sẽ xoá khỏi DOM sau khi đóng. Không truyền thì mặc định là false và modal không bị xoá khỏi DOM sau khi đóng
destroy() Tương tự với close(true)
buttonsEdit() Hỗ trợ chỉnh sửa các button trong modal bằng cách xác định button cần chỉnh sửa thông qua tham số index. Người dùng có thể cập nhật label, cssName, hoặc các callback hành động của nút nếu cần.
footerContentEdit() Cho phép chỉnh sửa nội dung của phần footer trong modal. Người dùng chỉ cần truyền nội dung mới thông qua tham số newContent, nội dung cũ sẽ được thay thế và hiển thị trực tiếp nếu modal đang mở.

Styling

class Opza.js Mô Tả
.opza Là lớp CSS dùng để định dạng modal bao phủ toàn bộ màn hình. Modal được căn giữa theo cả chiều ngang và chiều dọc, có nền tối mờ (rgba) tạo hiệu ứng làm nổi bật nội dung bên trong. Mặc định modal bị ẩn với opacity: 0 và visibility: hidden, đồng thời hỗ trợ hiệu ứng chuyển đổi mượt mà khi hiển thị thông qua thuộc tính transition.
.opza--show Là lớp CSS được thêm vào modal khi hiển thị. Lớp này đặt opacity: 1 và visibility: visible để làm modal xuất hiện trên màn hình, kết hợp với hiệu ứng chuyển đổi mượt mà từ lớp .opza.
.opza__box Là lớp CSS cho hộp chứa nội dung chính của modal. Lớp này có nền trắng, bo góc, bóng đổ nhẹ, và độ rộng cố định 500px. Khi xuất hiện, nó được mở rộng từ trạng thái thu nhỏ nhờ hiệu ứng transform: scale.
.opza__btn-close Là lớp CSS dành cho nút đóng của modal. Nút này được định vị ở góc trên cùng bên phải, có hình dạng tròn với kích thước nhỏ gọn, nền màu nhạt, và chữ màu xám.
.opza__content Là lớp CSS dành cho phần nội dung chính của modal. Phần này có khoảng cách đệm phía dưới, chiều cao tối đa 80% màn hình và hỗ trợ cuộn nội dung.
.opza__footerContent Là lớp CSS dành cho phần nội dung footer của modal, có đường viền trên mảnh, khoảng cách đệm gọn gàng.
.opza__buttons Là lớp CSS dành cho phần chứa các nút trong modal, sử dụng flexbox để căn chỉnh các nút ở bên phải và tạo khoảng cách giữa các nút.

HD sử dụng

javascript 
const modal = new Opza({
  // templateId: "modal-1",         
  content: `<div>Nội dung content của modal</div>`,
  widthModal: '600px' // Không truyền mặc định modal rộng 500px (nhận string)
  cssContainer: ["modal__content", "class-N"], // Tự đặt tên class cho thẻ chứa content bên trên
  closeTrigger: ["button", "overlay", "Escape"], // Mặc định có 3 cách để đóng modal.
  // Tắt bằng cách xoá nội dung tương ứng trong mảng
  scrollLock: false, // Truyền false nếu muốn tắt tính năng khoá cuộn trang khi modal đang bật
  destroy: true, // Nếu muốn modal xoá khỏi DOM sau khi đóng
  footerContent: "<p>Đây là nội dung footer</p>",
  buttons: [ // mảng chứa các object nút button
    { // Nút thứ nhất
      label: "Huỷ Bỏ",
      cssName: "modal__btn btn--outline",
      preAction: () => { // Có thể bỏ qua
        console.log("Kiểm tra trước khi Huỷ Bỏ");
        return true; // Nếu false, sẽ không chạy action
      },
      action: () => { // Có thể bỏ qua
        console.log("Xử lý logic khi bấm vào Huỷ bỏ");
        modal.close();
      },
    },
    { // Nút thứ 2
      label: "Xác Nhận",
      cssName: "modal__btn btn--primary",
      preAction: () => { // Có thể bỏ qua
        console.log("Kiểm tra điều kiện trước khi xác nhận");
        return confirm("Bạn có chắc chắn muốn xác nhận không?");
      },
      action: () => { // Có thể bỏ qua
        console.log("Xử lý logic khi bấm vào Xác nhận");
        modal.close();
      },
    },
  ],
});

// Chỉnh sửa nội dung footerContent
modal.footerContentEdit("Nội dung mới của footerContent");

// Chỉnh sửa các nút button
modal.buttonsEdit(
        0, // Xác định button cần sửa bằng index. Bắt đầu là 0, tính từ trái -> phải
        "newLabel",
        "modal__btn btn--danger",
        () => {
          console.log(
            "PreAction đã được gọi trước khi thực hiện hành động chính"
          );
        }, // preAction (callback trước khi hành động chính)
        () => {
          console.log("Action đã được thực hiện!");
          modal.close(); // Ví dụ là đóng modal sau khi thực hiện hành động chính
        }
      );
 
// Mở modal
modal.open();    

// Đóng modal
modal.close(); // Truyền true nếu muốn modal xoá khỏi DOM sau khi đóng
